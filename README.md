# Lotus
![CI](https://github.com/alexpyoung/lotus/workflows/CI/badge.svg)

## Development
### Installation
```
yarn install
```

### Build
```
bundle install
bundle exec fastlane ios build
```

### Test
```
bundle install
bundle exec fastlane ios test
```

### Add Dependency
```
bundle install
bundle exec fastlane ios install
```
